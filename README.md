# Gtk3 Auto Dark

Automatically switch Gtk3 applications to dark mode.

## Installation

Download the latest ZIP file from [releases], and install with

$ gnome-extensions install gtk3-auto-dark@swsnr.de.shell-extension.zip

Alternatively, install from [extensions.gnome.org](https://extensions.gnome.org/extension/7547/gtk3-auto-dark/), but note that releases on extensions.gnome.org may be delayed or outright rejected by its mandatory review process.
The author of this extension does not use extensions.gnome.org.

[releases]: https://codeberg.org/swsnr/gnome-shell-extension-gtk3-auto-dark/releases
